# Algorytmy

## Opis projektu

Celem projektu jest przedstawienie oraz omówienie najważniejszych algorytmów wykorzystywanych w informatyce i programowaniu. Projekt ma charakter edukacyjny i służy do nauki zasad działania algorytmów, ich implementacji oraz analizy efektywności. 
W projekcie znajdują się przykłady algorytmów wraz z opisem działania, analizą złożoności obliczeniowej oraz przykładowymi implementacjami w wybranym języku programowania.
